# 🌱 Fertilizer Recommendation Model

A machine learning-based approach to recommend fertilizers using multiple classification algorithms. This project focuses on comparing model performance rather than deployment. Models are compared using accuracy and other relevant evaluation metrics.

---

## 📌 Overview

This notebook explores how different machine learning models perform in predicting suitable fertilizers based on input features.

The goal is to:

* Train multiple models
* Evaluate their performance
* Compare accuracy and effectiveness

---

## ⚙️ Models Used

* 🔹 Support Vector Machine (SVM)
* 🌲 Random Forest
* ⚡ XGBoost
* 📊 Logistic Regression
* 🧠 Artificial Neural Network (ANN)

---

## 📊 Workflow

1. Data preprocessing
2. Feature selection / encoding
3. Model training
4. Accuracy evaluation
5. Model comparison

---

## 📈 Results

Each model was evaluated based on accuracy. The notebook includes:

* Model-wise accuracy comparison
* Observations on performance
* Insights into which algorithm performs best
